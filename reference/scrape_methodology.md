# For a provided DfT URL, return the documents that have dynamically updating methodology links.

For a provided DfT URL, return the documents that have dynamically
updating methodology links.

## Usage

``` r
scrape_methodology(url)
```

## Arguments

- url:

  URL of the DfT web page to scrape URLs from.

## Value

A table containing two columns, the collection URL and the methodology
URL.
