# For a provided ORR url, return those stats which have dynamically updating methodology links

For a provided ORR url, return those stats which have dynamically
updating methodology links

## Usage

``` r
scrape_orr_methodology(url)
```

## Arguments

- url:

  URL of the ORR web page to scrape urls from

## Value

a table containing two columns, the collection url and the methodology
url
