# For a given vatiable, check if it includes year information

For a given vatiable, check if it includes year information

## Usage

``` r
year_column_check(col)
```

## Arguments

- col:

  Name of the variable to check

## Value

Returns a true or false value as to whether the variable contains year
data, and meets the checks included. Checks include if the length of the
value is 4, numeric and starts with '19' or '20'.
