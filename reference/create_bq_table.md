# For a provided table, create and upload a table to BigQuery

For a provided table, create and upload a table to BigQuery

## Usage

``` r
create_bq_table(x, name, file_name)
```

## Arguments

- x:

  Name of dataset

- name:

  Name of sheet in workbook

- file_name:

  Name of workbook

## Value

The creation and uploading of a tidied dataset in BigQuery
